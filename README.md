## Getting started
In order to get started, create a .env file and provide the following environment variables
```
MYSQL_ROOT_PASSWORD=
MYSQL_DATABASE=
MYSQL_USER=
MYSQL_PASSWORD=
DB_URL=jdbc:mysql://yourMysqlServiceName:3306/yourDbName
```
